# Task 7

Execute the python script fill_database.py to generate content for a given database.

```
usage: fill_database.py [-h] [--seed SEED] --country_count COUNTRY_COUNT --player_count PLAYER_COUNT --olympics_count OLYMPICS_COUNT --event_count EVENT_COUNT url

positional arguments:
  url                   Used database URL.

optional arguments:
  -h, --help            show this help message and exit
  --seed SEED           Random seed for reproducibility.
  --country_count COUNTRY_COUNT
                        Number of countries to be generated.
  --player_count PLAYER_COUNT
                        Number of players to be generated.
  --olympics_count OLYMPICS_COUNT
                        Number of olympics to be generated.
  --event_count EVENT_COUNT
                        Number of events to be generated.
```
